🚀 ONE-CLICK STARTUP (EASIEST WAY)

1. Double-click: START_LEGALMITRA.vbs
   (or START_LEGALMITRA.bat if .vbs doesn't work)

   That's it! The script will:
   ✓ Check Python installation
   ✓ Create virtual environment  
   ✓ Install dependencies
   ✓ Setup configuration
   ✓ Start the server

2. On first run, you'll be asked to add your API key
   - Get OpenAI key: https://platform.openai.com/api-keys
   - Or get Anthropic key: https://console.anthropic.com/
   - Edit backend\.env file

3. Access the application:
   - Backend API: http://localhost:8888
   - API Docs: http://localhost:8888/docs  
   - Frontend: Open frontend\index.html in browser
   
   Note: Using port 8888 by default (change in backend\.env if needed)

📋 PREREQUISITES

• Python 3.8 or higher
  Download: https://www.python.org/
  ⚠ IMPORTANT: Check "Add Python to PATH" during installation

• API Key (one of):
  - OpenAI: https://platform.openai.com/api-keys
  - Anthropic: https://console.anthropic.com/
