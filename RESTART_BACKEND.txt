============================================
  HOW TO RESTART BACKEND WITH NEW MODELS
============================================

The backend needs to be restarted to load the new models (33 models including 6 FREE ones!)

METHOD 1: Quick Restart
-----------------------
1. Go to the command prompt where backend is running
2. Press Ctrl+C (stops the backend)
3. Wait 2 seconds
4. Press UP arrow key (recalls last command)
5. Press ENTER
6. Backend restarts with new models!

METHOD 2: Manual Restart
------------------------
1. Go to backend terminal
2. Press Ctrl+C
3. Type: python -m app.main
4. Press ENTER

METHOD 3: New Terminal
----------------------
1. Open NEW Command Prompt
2. Type: cd D:\SanMitra_Tech\LegalMitra\backend
3. Type: python -m app.main
4. Press ENTER
