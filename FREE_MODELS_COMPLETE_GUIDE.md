### How FREE Models Work:

1. **OpenRouter's FREE tier**
   - Some providers offer FREE access via OpenRouter
   - Limited to certain models
   - Sponsored by providers (Meta, Google, etc.)

2. **Model IDs format**
   ```
   provider/model-name:free
   â”‚       â”‚          â”‚
   â”‚       â”‚          â””â”€ Free tier indicator
   â”‚       â””â”€ Specific model version
   â””â”€ Provider (meta-llama, google, etc.)
   ```

3. **Context length**
   - How much text the model can process at once
   - Measured in tokens (roughly 4 chars = 1 token)
   - Examples:
     - 16k = ~64 pages
     - 128k = ~512 pages
     - 1M = ~4000 pages!

4. **Parameters (70B, 27B, etc.)**
   - Number of neural network parameters
   - More parameters = smarter (usually)
   - But also slower
   - 3B = very fast, good for simple tasks
   - 70B = slower, great for complex tasks
